## You open a website -> Terms and Conditions page pops up -> you click "Accept"
Aaaaand, there you are, potentially vulenerable to signing you newborn away!
Why? because you don't know what that long piece of text holds, because you didn't read it -- and no one does for understandable reasons.

A python program to analyze Terms and Conditions from a webisite and warn the user against the dangerous clauses it holds.
